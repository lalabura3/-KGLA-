> **产品名称：** 学知图谱 (LearnGraph)  
> **产品定位：** AI 驱动的视频学习知识图谱工具  
> **项目状态：** ✅ MVP 交付验收通过（综合评分 8.3/10）  
> **整理日期：** 2026-05-01

---

## 一、产品概述

### 1.1 愿景
让每一段视频学习过程变得可感知、可追溯、可交互。通过 AI 自动将视频内容结构化，生成可浏览的知识网络，帮助从"看视频"升级为"构建知识体系"。

### 1.2 目标用户
| 用户 | 核心需求 |
|------|---------|
| 学生/自学者 | 快速回顾视频重点，理清知识点关系 |
| 教师/讲师 | 上传教学视频自动生成笔记和知识结构 |
| 研究人员 | 快速定位关键知识点，建立知识关联 |

### 1.3 核心价值
- **从看到懂：** AI 自动提取知识点，将线性视频转化为结构化知识网络
- **从找到查：** 每个知识点绑定时间戳，点击即可跳转视频位置
- **从散到连：** 展示知识点间关联关系，构建完整知识图谱

---

## 二、核心用户流程

```
导入视频 → AI处理(ASR+LLM) → 查看笔记 → 探索图谱
```

1. **导入阶段：** 上传视频文件或输入公开视频链接
2. **处理阶段：** 自动 ASR → 分段 → LLM摘要 → 知识点提取 → 关系构建
3. **学习阶段：** 查看 AI 生成的图文笔记（绑定时间戳）
4. **探索阶段：** 知识图谱中浏览节点，点击查看详情并跳转视频
5. **深化阶段(P1)：** 编辑笔记、搜索图谱、AI问答、标记掌握度

## 五、图谱交互模式

| 模式 | 触发 | 行为 |
|------|------|------|
| **章节聚类** | 默认 | 根据分段章节将知识点聚类显示 |
| **聚焦模式** | 点击节点 | 以选中节点为中心展开1-2跳邻居，其余半透明 |
| **路径模式** | 选择两个节点 | 展示两节点间最短关联路径 |

### 视觉规范
- 主色调：蓝紫色 `#0D8CE0`
- 节点大小：关联重要度 0.0-1.0
- 节点颜色：按知识类型区分（概念/术语/公式/方法/例子/人物/事件）
- 边线粗细：关系权重 0.0-1.0

## 七、数据模型

```
Video ---< NoteSegment (多个段落)
  |          |
  |          +--- timestamp, title, summary, content
  |
  +--- KnowledgeNode (多个节点)
  |        |
  |        +--- id, name, description, type, importance, timestamp
  |
  +--- Relation (多条关系边)
           |
           +--- source_node → target_node, relation_type, weight
```
